## Data analysis notebooks for COMM3180 Group Data Project

* This folder should contain the a series of Jupyter notebooks you create to do the data analysis.

For the Business Schools notebook, I analyzed three datasets: one on full-time position percentage data for Wharton and two datasets comparing mean starting salaries for both Wharton and Gies' Accounting, Finance, and Marketing majors across 2021-2025.

I analyzed all datasets in a similar way: visually graphing the trends in income for both schools and comparing and contrasting across majors. While Wharton clearly has the edge in starting salary, Gies is slowly moving up in all majors while some Wharton majors are decreasing in salary or hitting a plateau (accounting and finance respectively). This is very interesting to see given that Gies is known for their accounting program.
